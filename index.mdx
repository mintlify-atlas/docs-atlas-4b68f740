  <Steps>
    <Step title="Create your account">
      Navigate to the login page and authenticate with your credentials. Playground uses Firebase Authentication for secure access.
      
      ```javascript
      // Firebase Authentication configuration
      const firebaseConfig = {
        apiKey: "YOUR_API_KEY",
        authDomain: "playgroundbdstop.firebaseapp.com",
        databaseURL: "https://playgroundbdstop-default-rtdb.firebaseio.com",
        projectId: "playgroundbdstop"
      };
      ```
    </Step>

    <Step title="Explore the dashboard">
      After logging in, you'll see the main dashboard with modules for Calculator, Herramientas (Tools), Plantillas (Templates), Procedimientos (Procedures), and Horarios Stop Jeans (Schedules).
      
      <Note>
        Your user preferences and theme settings are automatically saved to localStorage for a personalized experience.
      </Note>
    </Step>

    <Step title="Customize your workspace">
      Visit the **Configuraciones** (Settings) page to customize themes, colors, and preferences. Choose from multiple pre-built themes or create your own color scheme.
      
      ```javascript
      // Theme customization example
      localStorage.setItem('timerActive', 'true');
      localStorage.setItem('temaActual', 'custom-theme');
      ```
    </Step>

    <Step title="Start using features">
      Access any module from the navigation menu. Each feature is designed to streamline your workflow:
      - **Calculator**: Grade calculations with percentage breakdowns
      - **Templates**: Create and manage reusable templates with color coding
      - **Schedules**: Manage shifts and employee schedules for Stop Jeans stores
      - **Procedures**: Document processes with a rich text editor
    </Step>
  </Steps>
</div>
