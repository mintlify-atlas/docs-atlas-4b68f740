## What is Playground?

Playground is a comprehensive productivity web application designed to streamline daily tasks and workflows. Built with Firebase authentication and real-time database integration, it offers a suite of productivity modules accessible through a unified platform.

<img src="/images/logo.png" alt="Playground Logo" className="rounded-lg" />

## Available Modules

Playground includes the following productivity modules:

<Steps>
  <Step title="Calculadora (Calculator)">
    Grade calculator with percentage-based calculations and minimum passing grade features
  </Step>
  <Step title="Herramientas (Tools)">
    Collection of utility tools for various tasks
  </Step>
  <Step title="Plantillas (Templates)">
    Pre-built templates for common workflows
  </Step>
  <Step title="Procedimientos (Procedures)">
    Step-by-step procedure guides and documentation
  </Step>
  <Step title="Horarios Stop Jeans">
    Scheduling system for Stop Jeans operations
  </Step>
  <Step title="BlackJack Game">
    Integrated gaming module for breaks
  </Step>
</Steps>

## Technology Stack

<AccordionGroup>
  <Accordion title="Frontend">
    - **HTML5** with semantic markup
    - **CSS3** with custom CSS variables for theming
    - **JavaScript (ES6+)** for client-side logic

  <Accordion title="Backend & Database">

  <Accordion title="Additional Libraries">

## Architecture Overview

Playground follows a modular architecture with shared components:

```
playground/
├── Authentication Layer (Firebase Auth)
├── Navigation Header (shared across modules)
├── Module Pages
│   ├── Calculadora
│   ├── Herramientas
│   ├── Plantillas
│   ├── Procedimientos
│   └── Horarios
├── Configuration System
├── Theme Engine (PreferenciasColores)
└── Notification System
```

## Security Features

<Warning>
Playground implements several security measures:
- Session timeout after 8 hours
- Firebase authentication integration
- Email format validation
- Protected routes requiring authentication
</Warning>
